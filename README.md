# setting-nestjs

---

### 프로젝트 실행

npm run start

---

### 프로젝트 설정

##### [ nest.js 프로젝트 생성 ]

npx @nestjs/cli@8.0 new .  
npm install typescript@4.7.4

<br />

##### [ nest.js 기초 설정 ]

npm install @nestjs/swagger@5 swagger-ui-express --save  
npm install pg@8.7.1 typeorm@0.2.45 @nestjs/typeorm@8.0.1 --save
